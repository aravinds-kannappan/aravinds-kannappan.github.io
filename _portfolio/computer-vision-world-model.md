Trained a diffusion world model on 737K+ frames from a visual environment, combining a VAE encoder for latent state representation with a CNN-based reward model and PPO agents for reinforcement learning.

**Key components:**
- VAE encoder/decoder for compact latent state representation
- Diffusion model for next-frame generation in latent space
- CNN reward model estimating reward signal from visual observations
- PPO agents trained end-to-end within the world model
